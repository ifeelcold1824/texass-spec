# 回合
=====================

## 剩余玩家将在回合内轮流行动直到所有剩余玩家下注金额相同

    回合开始时系统的操作
    |round      |system action
    |-----------|-------------------
    |1st        |每个玩家发两张牌
    |2nd        |翻开三张底牌
    |3rd        |翻开第四张底牌
    |4th        |翻开第五张底牌

    玩家按如下行动，BET金额是玩家目前的下注总金额
    |action   |player1  |player2  |player3  |
    |---------|---------|---------|---------|
    |1        |BET:1    |         |         |
    |2        |         |BET:2    |         |
    |3        |         |         |BET:2    |
    |4        |BET:3    |         |         |
    |5        |         |BET:3    |         |
    |6        |         |         |BET:3    |

* 以用例中的三名玩家开始一个回合
* 系统进行当前回合所示的操作
* 剩余玩家按用例所示进行行动
* 回合将会结束
